Static portfolios are a thing of the past. They present a one-sided, lifeless view of a developer's journey. I decided to build this **AI-Portfolio** to break that mold.

This project is an **AI-powered, interactive portfolio** that transforms the "About Me" page into a dynamic, engaging conversation. ### **Installation & Usage** üöÄ

Want to get this project running on your local machine? Follow these simple steps.

#### **Prerequisites**
-   Node.js (v18 or higher)
-   pnpm package manager
-   Google Gemini API Key

#### **Local Setup**

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/sahilmishra108/Sahil-s-Portfolio.git
    cd Sahil-s-Portfolio
    ```

2.  **Install dependencies:**
    ```sh
    pnpm install
    ```

3.  **Set up your environment variables:**
    Create a `.env.local` file in the root of your project and add the following keys:
    ```env
    GOOGLE_GENERATIVE_AI_API_KEY="your_gemini_api_key_here"
    ```
    *   Get your Gemini API Key from [aistudio.google.com](https://aistudio.google.com/).

4.  **Run the development server:**
    ```sh
    pnpm dev
    ```

5.  **Open your browser** and navigate to `http://localhost:3000`.
